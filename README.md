# AES_POC

## Reviewer Notes

The code created for this submission is in `artifacts/aes_triage_poc.ipynb`.

The notebook is a solar-only proof of concept for renewable asset performance triage. It walks through synthetic asset and telemetry generation, a Performance Ratio (PR) based expected-generation baseline, sustained anomaly detection, rule-based root-cause attribution, financial impact ranking, validation against injected ground-truth faults, and figures supporting the analysis.
